# Practice

Создать объект с такой структурой:
```js
  obj = {
    x: 10,
    y: 20,
    inner: {
        x: 20,
        z: 30
    },
    foo2: {
        k: 23,
        p: 13,
    },
  };
```

Написать функцию convert(obj), который будет хранится в window,
он получает аргументом obj. Функция должна вернуть объект:

```js
newObj = {
  x: 20,
  y: 20,
  z: 30,
  k: 23,
  p: 13,
};
```
