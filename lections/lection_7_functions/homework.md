## Задача 2:

Написать функцию, которая примет как аргументы (параметры) два массива и сравнит суммы всех ЧИСЛОВЫХ элементов. Тот массив, сумма которого большая - должен вернутся функцией.

```js
let arr = [1, 'a', 2];

getMaxSumArr(arr, arr2); // -> arr
```
